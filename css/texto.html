Texto, cadenas y listas

Texto

En CSS podemos definir cómo realizar la manipulación de elementos de texto como los saltos de línea, la justificación, la alineación, la gestión de espacios en blanco y las transformaciones de texto.

text-indent

Sangrado, sangría o indentación. Sólo afecta a la primera línea del párrafo.

Los valores se indican con procentajes o medidas.

text-align

Establece la alineación del contenido de un elemento de bloque.

  • left
  • right
  • center
  • justify

text-transform

Transforma el texto de un elemento para mostrarlo en mayúsculas, minúsculas o una mezcla de ambas.

  • capitalize → Primera letra de cada palabra en mayúscula.
  • uppercase
  • lowercase
  • none → Muestra el texto tal y como está escrito.

text-decoration

Establece la decoración que se añade al texto de un elemento. Se puede utilizar varias decoraciones diferentes en el mismo elemento.

  • none
  • underline
  • overline
  • line-through
  • blink

Modificación de espacio

  • letter-spacing → Define la separación entre los caracteres de texto.
  • word-spacing → Define la separación entre las palabras del texto. El espacio suele ser 1em.

vertical-align

La propiedad vertical-align de CSS especifica el alineado vertical de un elemento en línea o una celda de una tabla.

  • baseline
  • middle
  • sub → Subíndice
  • super → Superíndice
  • text-top
  • text-bottom
  • top
  • bottom

Cadenas

Podemos poner en CSS cadenas de texto con comillas:

  • Dobles → "Esto es una cadena"
  • Simples → 'Esto es una cadena'
  • Alternadas vs \
    • "Esto es una 'cadena' "
    • 'Esto es una "cadena" '
    • "Esto es una \"cadena\" "
    • 'Esto es una \'cadena\' '

Para poner un salto de línea debemos indicarlo poniendo '\'

Si queremos poner una URI, lo indicaremos así: url(Uniform Resource Identifier)

Listas

list-style-type

Establece el tipo de viñetas para la lista.

  • Ordenadas → lower-roman, upper-roman, decimal, decimal-leading-zero, upper-alpha, lower-alpha, hebrew, armenian...
  • No ordenadas → disc, circle, square

list-style-image

Te permite usar una imagen personalizada para la viñeta, en lugar de un simple cuadrado o círculo.

list-style-position

Establece si las viñetas aparecen dentro de los elementos de la lista o fuera de ellos, antes del inicio de cada elemento.

  • outside
  • inside

list-style

Es la propiedad resumen de list. Podemos indicar las tres propiedaades anteriores en una única indicándolas una tras otra.

ul {list-style: square url(example.png) inside;}
