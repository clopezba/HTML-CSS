Formato de HTML

El formato básico que presenta un documento HTML es el que se muestra en la figura.

  • En HTML5 es obligatorio indicar la declaración del tipo de documento en la primera línea.
  • La etiqueta html indica el inicio y final del documento.
  • El primer elemento de HTML es head, es opcional pero recomendado ponerlo. Incluye los metadatos, información sobre el documento.
  • Body es el cuerpo del documento. Todo el contenido que se mostrará se incluye aquí.
