Tablas

Una tabla es un conjunto estructurado de datos distribuidos en filas y columnas. Permite buscar con rapidez y facilidad valores entre diferentes tipos de datos que indiquen algún tipo de conexión.

El elemento principal para insertar una tabla es ‹table›, que podrá contener los siguientes elementos:

  • Fila (table row): ‹tr›‹/tr›
  • Columna (table data - celda): ‹td›‹/td›
  • Títulos tabla (table header): ‹th›‹/th› → Lo pone en negrita. Suelen ser los encabezados de las filas o columnas.
  • Encabezado tabla: ‹thead›‹/thead›
  • Cuerpo tabla: ‹tbody›‹/tbody›
  • Sumario tabla: ‹tfoot›‹/tfoot›
  • Leyenda: ‹caption›‹/caption› → Si hay leyenda, debe ser el primer elemento hijo de la tabla.

Atributos tablas

De forma predeterminada la tabla se ajusta automáticamente al contenido y no tiene bordes. Esto se puede modificar con algunos de los atributos que admite o con las hojas de estilo CSS.

  • Borde (en table): border
  • Expansión de columna (en td y th): colspan sirve para combinar celdas de una misma fila.
  • Expansión de fila (en td y th): rowspan sirve para combinar celdas de una misma columna.
