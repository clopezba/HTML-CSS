Los elementos de agrupación

En HTML existen una serie de etiquetas que tienen como función agrupar y organizar la información. A continuación veremos las etiquetas más utilizadas con este propósito.

Agrupaciones

Las siguientes etiquetas se utilizan para agrupar información:

Párrafo (paragraph)

Se indica con la etiqueta ‹p›. Define un párrafo de texto.
El elemento p no debería utilizarse si otro elemento más específico es más apropiado.

‹p›Este texto pertenece a un párrafo‹/p›
‹p›Éste es el párrafo siguiente‹/p›

Texto preformateado (preformatted)

El elemento ‹pre› representa un bloque de texto preformateado. El texto en este elemento típicamente se muestra en una fuente fija, no proporcional, exactamente como es mostrado en el archivo. Los "blancos" dentro de este elemento también son mostrados como están escritos, repetando los espacios y los saltos de línea. Se podría utilizar, por ejemplo, para escribir un poema con una métrica definida.

‹pre›¿Qué es poesía?, dices mientras clavas
          en mi pupila tu pupila azul.
        ¿Qué es poesía?¿Y tú me lo preguntas?
          Poesía... eres tú.
                            -Gustavo Adolfo Bécquer‹/pre›
¿Qué es poesía?, dices mientras clavas
      en mi pupila tu pupila azul.
    ¿Qué es poesía?¿Y tú me lo preguntas?

Línea horizontal (horizontal rule)

El elemento ‹hr› representa un cambio de tema entre párrafos (por ejemplo, un cambio de escena en una historia, un cambio de tema dentro de una sección, etc.).
Entre secciones no sería necesario porque ya lo indican los títulos.

‹p›Éste es el primer párrafo.‹/p›
‹hr›
‹p›Éste es el segundo párrafo, separado del primero por una línea horizontal.

Listas

En HTML se pueden crear listas de información de una forma sencilla. Encontramos dos posibilidades: listas ordenadas y listas desordendas.

Listas desordenadas (unordered list)

Define una lista de items, en el que su orden no es importante, es decir, si cambiamos el orden no cambiará el sentido del documento. Se trata de una etiqueta contenedora que se indica con la etiqueta ‹ul›.

‹ul›
  ‹li›Esto‹/li›
  ‹li›Lo otro‹/li›
  ‹li›Lo de más allá‹/li›
‹/ul›

  • Esto
  • Lo otro
  • Lo de más allá

Listas ordenadas (ordered list)

El elemento ‹ol› permite definir listas o niñetas ordenadas, bien con numeración o alfabéticamente.

Argumentos de ‹ol›

Reversed: ‹ol reversed›
Se utiliza para revertir en orden de los elementos. La enumeración será en orden descendente.

Start: ‹ol start="valor"›
Indica por qué valor empieza la lista. En el caso de ejemplo empezaría por 4:

‹ol start="4"›

Elementos de lista (list item)

El elemento ‹li› declara uno de los elementos de una lista. Se debe poner un ‹li› por cada elemento.
Este elemento li admite, a su vez, dos atributos: tipo y valor.

Tipo: ‹li type="tipo"›
Se puede definir el tipo de enumeración que tendrá un elemento de forma aislada. Los diferentes tipos son: disc / square / circle (si no queremos dar un orden); 1 / a / A / i / I (para marcar un orden).

Valor: ‹li value="valor"›
Se utiliza para dar un valor concreto a un elemento expecífico. Los elementos posteriores seguiran a partir de este nuevo valor.

  ‹li›Ejemplo‹/li›
  ‹li value="7"›Ejemplo‹/li›

Listas de descripciones (description list)

El elemento ‹dl› encierra una lista de grupos de términos y descripciones. Algunos usos comunes para este elemento son implementar un glosario o para desplegar metadatos.

En este tipo de listas no se utiliza el elemento ‹li›, sino que, al tratarse de asociaciones de pares 'término-descripción', utilizaremos los elementos que se muestras a continuación:

Término de la descripción (description term)

Este elemento (‹dt›) especifica un término en una descripción o lista de definiciones. Usualmente va seguido por un elemento ‹dd›; sin embargo, múltiples elementos ‹dt› en un renglón indican diferentes términos los cuales todos son definidos por el siguiente elemento ‹dd›.

Datos de la descripción (description data)

El elemento ‹dd› provee la definición u otro texto relacionado que es asociado con el término precedente (‹dt›).

En una lista de descripciones, de forma predeterminada aparecen los datos de las descripción con sangrado a la izquierda.
